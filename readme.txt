The Parallax Gallery ACF Addon elevates the presentation of your ACF gallery field images, turning them into a captivating parallax image gallery for a more visually striking experience.

Installation

Download the plugin zip file.
Go to your WordPress dashboard.
Navigate to Plugins -> Add New.
Click on the "Upload Plugin" button.
Upload the downloaded zip file and activate the plugin.

Usage

Make sure ACF PRO is installed and activated.
Activate the Parallax Gallery ACF Addon.
Create a new gallery field named "acf_gallery" using ACF PRO.
Show gallery using shortcode[ParallaxGalleryACF].
